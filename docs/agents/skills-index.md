# Matt Pocock Skills Index (HAM Repo-Local)

This repo installs a curated local subset adapted from `mattpocock/skills` (MIT).

## Active Skills

- `setup-matt-pocock-skills`: verify/update local skill wiring docs in `docs/agents/`.
- `grill-with-docs`: align on larger changes and update shared language/ADRs.
- `diagnose`: structured bug/perf regression loop.
- `tdd`: red/green/refactor for focused fixes.
- `zoom-out`: understand a subsystem before edits.
- `to-issues`: break accepted plans into vertical slices.
- `to-prd`: turn chat intent into a clear implementation spec.
- `handoff`: compact long threads into restartable context.
- `ham-qa-prompt-gate`: review implementation prompts for red flags/alignment/blockers/scope drift/evidence gaps, rewrite when fixable, and stop when human decisions are required.

## Parked / Manual-Only Skills

- `triage`: optional issue hygiene only when explicitly requested; manual-only and not default during active Builder stabilization.
- `improve-codebase-architecture`: architecture cleanup only; manual-only and parked during active Builder stabilization.
